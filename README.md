# 🧠 Deepfake Detection System

A Deep Learning based system that detects whether a given face image is **Real or Fake (Deepfake)**.

The project processes videos to extract frames, detects faces, trains a CNN model, and provides a **Streamlit web interface** where users can upload images to check if they are real or deepfake.

---

# 🚀 Features

- Video to Frame Extraction
- Face Detection using MTCNN
- CNN based Deepfake Detection
- Image Upload Interface
- Real vs Fake Prediction
- Streamlit Web Application

---

# 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- OpenCV
- MTCNN
- NumPy
- Streamlit
